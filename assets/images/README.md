# Images Folder

Place your custom images here for the Igrix website.

## Recommended Images:

- **hero-bg.jpg** - Hero section background (1920x1080px)
- **team-collaboration.jpg** - Team working together (800x600px)
- **web-development.jpg** - Coding/development images (800x600px)
- **ui-design.jpg** - Design mockups (800x600px)
- **api-integration.jpg** - Technical diagrams (800x600px)
- **logo.png** - Company logo (transparent background)
- **favicon.ico** - Browser favicon (32x32px)

## Image Optimization Tips:

1. Compress images before uploading (use tools like TinyPNG)
2. Use WebP format for better performance
3. Keep file sizes under 500KB for web images
4. Use descriptive filenames (e.g., hero-web-development.jpg)
5. Maintain aspect ratios for consistency

## Current Setup:

The website currently uses Unsplash images via CDN. Replace these URLs in index.html with your local images:

```html
<!-- Example -->
<img src="assets/images/hero-bg.jpg" alt="Hero Background">
```
